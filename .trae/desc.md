1. 背景
```text
目前市场关于 nginx 管理工具比较流行的是 nginx-ui、manager nginx proxy 等软件，但是有个问题是，他们只能管理单个机器上的 nginx，如果要管理多个机器上的 nginx，则需要在每台机器上部署一个 nginx-ui 或 manager nginx proxy，当机器数量很多时，会带来一定的成本。
```

2. 个人角色
```text
Linux 运维工程师
```

3. 项目进展
```text
mngxops 是我目前正在开发的一个基于 django 的 nginx 管理工具，它的目标是提供一个简单、易用的界面，来管理多个 nginx 服务器。
但是目前还在开发中，有很多功能需要实现，比如角色权限、日志管理、配置管理等。
由于是个人项目，而且是一个运维工程师。没有专业开发、前端、后端、产品功能设计、页面样式设计等能力，所以项目目前功能都是类似东拼西凑的形式。而且很多地方都需要优化，比如代码质量、性能优化、用户体验、相同的功能在页面展示样式不同等。
```
