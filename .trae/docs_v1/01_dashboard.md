# MngxOps 需求文档 - 01 仪表盘

> **功能模块**: 仪表盘（Dashboard）  
> **URL**: `/`  
> **视图**: `apps/dashboard/views.py` → `index()`  
> **模板**: `apps/dashboard/templates/dashboard/index.html`

---

## 1. 功能概述

仪表盘是用户登录后的首页，提供系统整体运行状况的快速概览。核心定位是"一屏看全貌"，让运维人员快速掌握节点健康度与近期操作动态。

---

## 2. 数据模型

仪表盘不独立建模，聚合展示以下模型统计数据：

| 模型 | 统计指标 |
|------|---------|
| `Node` | 总节点数、在线数、离线数、最近操作列表 |
| `NodeGroup` | 节点组总数 |
| `ConfigVersion` | 配置版本总数 |
| `ReleaseTask` | 发布任务总数、最近发布任务列表 |
| `Config` | 同步失败的配置列表 |

### 3.2 详细元素说明

#### 3.2.1 统计卡片区域

- **排列**: 响应式网格，大屏 4 列，中屏 3 列，小屏 2 列
- **样式**: Bootstrap Card + 图标 + 数字 + 标签
- **颜色语义**:
  - 在线节点卡片：绿色边框 / 绿色数字
  - 离线节点卡片：红色边框 / 红色数字
  - 同步失败卡片：橙色/黄色边框
  - 其他卡片：蓝色边框 / 默认色
- **交互**: 在线/离线/同步失败卡片可点击跳转到对应列表页（带筛选条件）

#### 3.2.2 最近操作节点列表

| 列名 | 说明 |
|------|------|
| 主机名 | `node.hostname`，可点击跳转节点详情 |
| IP地址 | `node.ip` |
| 状态 | 在线(绿) / 离线(红) / 未知(灰) 徽标 |
| 环境 | 开发/测试/生产 徽标 |
| Nginx版本 | `node.nginx_version` 或 "未获取" |
| 更新时间 | `node.updated_at`，相对时间显示（如"3分钟前"） |

#### 3.2.3 最近发布任务列表

| 列名 | 说明 |
|------|------|
| 批次号 | `task.batch_number` |
| 配置名称 | `task.config.name`，可点击跳转配置详情 |
| 版本 | `task.version.version` |
| 目标节点 | `task.node.hostname` |
| 操作人 | `task.operator.username` |
| 状态 | 成功(绿)/失败(红)/进行中(蓝)/等待中(灰) 徽标 |
| 创建时间 | `task.created_at`，相对时间显示 |

#### 3.2.4 同步失败配置告警

| 列名 | 说明 |
|------|------|
| 配置名称 | `config.name`，可点击跳转配置详情 |
| 关联节点 | 节点 hostname 列表 |
| 同步状态 | "同步失败" 红色徽标 |
| 最后同步 | `config.last_sync_time`，相对时间 |
| 错误信息 | `config.last_sync_error`，截断显示(最多50字符)，hover 显示完整内容 |
| 操作 | "重试同步" 按钮 / "查看详情" 链接 |

## 4. 业务规则

| 编号 | 规则 | 说明 |
|------|------|------|
| R1 | 数据刷新 | 页面数据为快照，不自动刷新；提供"刷新"按钮手动刷新 |
| R2 | 空状态 | 无数据时显示友好的空状态提示，如"暂无节点，去添加" |
| R3 | 权限控制 | 仅有对应模块 read 权限时显示相关卡片和列表 |
| R4 | 数量限制 | 最近列表最多显示 10 条 |

---

## 5. 交互行为

| 交互 | 触发 | 行为 |
|------|------|------|
| 点击统计卡片 | 点击在线/离线/失败数字卡片 | 跳转到对应列表页面，自动加上对应筛选条件 |
| 点击节点名称 | 点击节点主机名 | 弹出节点详情弹窗（含系统信息） |
| 点击刷新按钮 | 点击页面顶部刷新图标 | 重新加载页面数据 |
| Toast提示 | 登录后首次进入 | 显示欢迎消息 + 上次登录信息 |

## 7. 重构建议

| 编号 | 问题 | 建议 |
|------|------|------|
| S1 | 现有仪表盘仅显示原始数据，缺乏可视化 | 增加节点在线率环形图、同步状态饼图 |
| S2 | 统计数据无趋势对比 | 增加"较昨日"变化量（↑3 / ↓1） |
| S3 | 最近列表占用空间大 | 改用 Tab 切换："最近节点""最近任务""告警" |
| S4 | 缺少快速操作入口 | 增加快捷操作按钮区（"添加节点""新建配置""发布"） |
